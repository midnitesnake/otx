Description
-----------

This is an up-to-date version of otx.

Following changes has been made:

* Based on 10.9 SDK.
* 64bit binary.
* Works with new otool came with Xcode 4.2+.
* Source code compatible with Xcode 5+.
* Tons of bug fixes.
